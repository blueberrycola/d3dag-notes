Sugiyama Layout Analysis:

It is used to render DAGs by assigning each node a layer. Aligning nodes within a layer to produce a good layout. Primary method used in d3-dag

About:
The sugiyama layout can be configured with different algorithms for each stage of the layout. For each stage there should be adecuate choices for methods that balance speed and quality for your needs
Choices:
Layering: assigns each node a layer. {Simplex (slow), Longest path (fast) coffman graham (‘medium’)}
OBSERVATION: in the document page selecting coffman causes a crash on notebook web app
Crossing Minimization: This sets how to arrange each node in a layer to minimize edge crossings, most expensive part.
Optimal contained the least edge crossings
two layer opt, median, and mean are the others.
OBSERVATION: browser switched through them quickly as opposed to layering drop down
Coordinate Assignment: This gives ordered nodes in a layer an actual x coord, spacing some out more than others
Quadratic: looks good but slow on performance
Greedy: looks ok (medium on performance)
Center: Several edge crossings, hard to read (fast performance)
